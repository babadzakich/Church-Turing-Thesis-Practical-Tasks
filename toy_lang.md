## Общая информация

Игрушечный язык — это простой императивный язык без функций и циклов, предназначенный для задач анализа и оптимизации программ.

Программа выполняется **последовательно сверху вниз**.


## Структура программы

Программа — это последовательность инструкций:

```c
stmt;
stmt;
stmt;
...
```

## Переменные

* Имена переменных:

  * начинаются с буквы или `_`
  * могут содержать буквы, цифры и `_`

Примеры:

```c
x
y1
_tmp
```

Все переменные только типа int64, поэтому имя типа при определении не используется.
Объявление переменной без определения запрещено.

## Типы данных

В языке есть два типа:

* **целые числа (int64)**
* **булевы значения (true / false)**


## Целочисленные выражения

### Константа

```c
5
-10
0
```

### Переменная

```c
x
```

### Сложение

```c
a + b
a + 5
3 + x
```

### Ввод

```c
read()
```

Возвращает произвольное целое число.


## Булевы выражения

### Константы

```c
true
false
```

### Сравнения

```c
a < b
x == 5
y != z
```

Допустимые операции:

```text
== != < > <= >=
```

Операнды — целочисленные выражения.


## Инструкции

### 1. Присваивание

```c
x = expr;
```

где `expr` — целочисленное выражение.

Примеры:

```c
x = 5;
y = x + 1;
z = read();
```


### 2. Условный оператор

```c
if (cond) {
    ...
} else {
    ...
}
```

Особенности:

* **всегда есть обе ветки**
* `cond` — булево выражение
* тело — список инструкций

Пример:

```c
if (x < 0) {
    y = 1;
} else {
    y = 2;
}
```


### 3. Вывод

```c
print(cond);
```

Выводит значение булевого выражения.

Примеры:

```c
print(true);
print(x < y);
```


## Семантика

### Выполнение

* программа выполняется последовательно
* инструкции внутри блока выполняются по порядку


### Переменные

* переменные могут переопределяться
* при входе в программу все переменные имеют значение **undef**


### Операции

* все вычисления выполняются в `int64`
* **переполнений не происходит** (гарантируется)


### read()

* возвращает произвольное целочисленное значение
* каждый вызов может возвращать разное значение

## Пример

```c
x = read();

if (x < 0) {
    if (x < -5) {
        y = 1;
    } else {
        y = 2;
    }
} else {
    y = 3;
}

print(y > 0);
```

## Гарантии входных данных

* синтаксис корректен
* все `if` имеют `else`
* переменные объявлять не нужно
* имена переменных корректны
* выражения корректны по типам
