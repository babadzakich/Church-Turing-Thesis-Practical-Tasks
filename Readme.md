# Задача 1. Построение CFG

## Условие

Дан текст программы на [игрушечном языке программирования](./toy_lang.md).

Необходимо:

1. распарсить программу;
2. построить для неё граф потока управления (CFG);
3. вывести CFG в каноническом текстовом формате.

Формат CFG строго фиксирован и описан в файле [./cfg_spec.md](./cfg_spec.md).

Все корректные решения должны выдавать **одинаковый CFG** для одной и той же программы.



## Входные данные

Путь к файлу с программой на игрушечном языке.

Программа содержит:
- присваивания
- условные операторы `if`
- вывод `print`



## Выходные данные

Текстовое представление CFG в формате из [./cfg_spec.md](./cfg_spec.md).

Вывод должен быть записан в файл: `<input_path>.ans`

## Пример

Вход:

```

x = 1;
y = 2;
if (x < y) {
    x = 3;
} else {
    y = 4;
}
z = x + y;
print(z > 0);

```

Выход:

```

block0:
x = 1;
y = 2;
if (x < y) goto block1; else goto block2;

block1:
x = 3;
goto block3;

block2:
y = 4;
goto block3;

block3:
z = x + y;
print(z > 0);
return;

```
