# Формула ЛП

## Условие 
Тебе дано условие "Напишите программу, которая по поданному выражению
определяют, является ли оно формулой логики предикатов или нет. 

Программа читает файл input.txt. 
В первой строке файла подаётся информация о предикатных символах и их местности через пробел. 

Предикатными символами могут быть только заглавные латинские буквы, кроме "A" и "E". 
Например, "P 1 R 2 Q 3". 

Во второй строке подаётся информация о функциональных символах
и их местностях через пробел. Функциональными символами могут быть только
строчные латинские буквы. 
Например, "f 2 g 2 h 1". 

В третьей строке подаются константные символы через запятую. 
Константными символами могут быть только строчные латинские буквы. 
Например, "c d e". 

Отдельно отметим, что использоваться будут только переменные вида x_1, x_2, x_3, ...
Переменных другого вида не будет. 

В формулах конъюнкция будет обознаться "&", дизъюнкция будет обозначаться "|", отрицание будет обозначаться "!",
импликация будет обозначаться ">", квантор существования будет обозначаеться "E", квантор всеобщности будет обозначаться "A". 

После этого в файле будут перечислены по одному на строку выражения, которые надо проверить на определение формулы логики предикатов. 

Свои ответы программа будет записывать в файл output.txt. Если поданное выражение является
формулой, то программа должна записать в файл ответ "YES", иначе - ответ "NO". По одному ответу на строку.
